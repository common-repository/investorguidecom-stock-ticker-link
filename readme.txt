== Description ==

This plugin automatically looks for ticker symbols like (AAPL) or (GOOG) and link the tickers to research pages at InvestorGuide.com

To set it up, just add the plugin to your wordpress installation and activate it.

Here's the link to our [plugin homepage](http://www.investorguide.com/article/6100/stock-ticker-link-wordpress-plugin/) so that you can see how this works.

== Installation ==

1. Upload 'investorguide-stock-ticker-links' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. When you write your article, include ticker symbols in parenthesis -- i.e., (AAPL) or (GOOG)
4. When the article is published, all tickers will be linked to the research page on InvestorGuide.com
